# LensClip PRD - 製品要件定義書

## 概要
LensClipは、小さい子供を持つ親向けの「これなぁに？」スクラップWebアプリ。
写真を撮るとAIが「これはなぁに？」の答えを返し、親子で一緒に学べる図鑑体験を提供する。

## ターゲットユーザー
- **プライマリ**: 3-6歳の子供を持つ親（操作者）
- **エンドユーザー**: 親子で一緒に使う

## 主要ユースケース
1. **撮影→同定**: 親が子供の発見をカメラで撮影→AIが「これはOOだよ」と教える
2. **英単語学習**: 同定結果の英名をTTSで発音、親子で英語に触れる
3. **振り返り**: 過去の発見をライブラリで見返す、タグやカテゴリで整理
4. **達成感**: 撮った枚数や種類の蓄積（バッジ等は未実装）

## 機能

### 撮影・アップロード
- カメラ or ファイル選択
- 位置情報取得（許可時）
- 画像圧縮（最大1024px幅、WebP 80%品質）、EXIF除去

### AI分析
- Vision API で主対象の bbox を検出 → 合成スコア（信頼度×面積×中心距離）で1件選定
- bbox でマージン付き crop → Gemini API で同定・説明生成
- 最大3候補のカード情報（名前、英名、子供向け説明、見分けポイント、豆知識）
- 非同期 Queue 処理 + SSE リアルタイム通知

### 英名読み上げ（TTS）
- Google Cloud Text-to-Speech API で同定結果の英名を発音
- 英単語学習の補助機能として、親子で英語に触れるきっかけを提供
- キャッシュ付き（テキスト+音声+速度のハッシュキー、7日TTL）

### ライブラリ
- グリッド表示、検索、タグフィルタ
- 表示モード: 日付 / カテゴリ / マップ

### タグ・カテゴリ
- タグ: AI自動付与 + 手動追加
- カテゴリ: AI自動分類 + 手動修正（`config/categories.php` 定義）

### 管理機能
- ログ閲覧（レベル・日付フィルタ）
- Geminiモデル切替（allowlist制御）

### 認証
- Laravel Breeze（メール/パスワード）
- Google OAuth（Socialite）

## 技術スタック
- **Backend**: Laravel 12 + MySQL
- **Frontend**: Inertia.js v2 + React 18 + TypeScript + Tailwind CSS v3
- **AI**: Google Cloud Vision API（Object Localization）+ Gemini API
- **TTS**: Google Cloud Text-to-Speech API
- **Storage**: GCS（本番）/ ローカル public（開発）
- **Queue**: Redis（本番）/ database（開発）
- **Dev Env**: Docker（Laravel Sail）
- **Deploy**: Railway

## 制約・配慮
- **セキュリティ**: APIキーはサーバーサイドのみ、EXIF除去
- **子供向け配慮**: 不適切コンテンツはブロック、短い説明で戻す導線
- **モバイル優先**: レスポンシブ、大きなタッチターゲット、片手操作前提

## UX / デザイン原則
- 写真 + アイコン + 状態表示を主体。テキストは最小限
- 撮影 → 結果表示まで CTA 2ステップ以内
- すべての操作に即時フィードバック（ローディング、成功、失敗）
- プレミアム感: 適切な余白、タイポグラフィ、マイクロアニメーション
- 親が操作し、AI の説明文は子供に読み聞かせる想定
- 未認証ユーザーは LP → 登録/ログイン（デモ導線なし）
