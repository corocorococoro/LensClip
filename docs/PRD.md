# LensClip PRD - 製品要件定義書

## 概要
LensClipは、小さい子供を持つ親向けの「これなぁに？」スクラップWebアプリ。
写真を撮るとAIが「これはなぁに？」の答えを返し、親子で一緒に学べる図鑑体験を提供する。

## ターゲットユーザー
- **プライマリ**: 3-6歳の子供を持つ親（操作者）
- **エンドユーザー**: 親子で一緒に使う

## 主要ユースケース
1. **撮影→同定**: 親が子供の発見をカメラで撮影→AIが「これはOOだよ」と教える
2. **振り返り**: 過去の発見をライブラリで見返す、タグやコレクションで整理
3. **達成感**: 撮った枚数や種類でバッジ獲得（MVP後）

## MVPスコープ

### ✅ 含む（MVP）
- ユーザー認証（ログイン/登録）
- 画像アップロード（カメラ/ファイル選択）
- AI分析（Vision API で主対象をCrop → Gemini で同定・説明）
- 結果表示（名前、子供向け説明、豆知識、安全注意）
- ライブラリ（グリッド表示、検索、タグフィルタ）
- タグ管理（自動付与 + 手動追加）
- コレクション（作成、追加、カバー設定）
- 処理状態管理（processing/ready/failed）
- 失敗時のリトライ

### ❌ 含まない（非スコープ）
- 通知機能
- 外部連携/SNS共有
- 画像加工（フィルタ/スタンプ等）
- マルチユーザー家族共有
- オフライン対応
- バッジ/達成システム（MVP後）

## 成功指標
- 1ユーザーあたり週5枚以上の撮影
- AI分析成功率 90%以上
- 処理時間 10秒以内（体感）

## 技術スタック
- **Backend**: Laravel 12 + SQLite
- **Frontend**: Inertia.js + React + Tailwind
- **AI**: Google Cloud Vision API（Object Localization）+ Gemini API
- **Storage**: ローカルディスク（public）

## 制約・配慮
- **セキュリティ**: APIキーはサーバーサイドのみ、EXIF除去
- **子供向け配慮**: 不適切コンテンツはブロック、短い説明で戻す導線
- **モバイル優先**: レスポンシブ、大きなタッチターゲット
